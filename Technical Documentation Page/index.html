Introduction

JavaScript is a high-level, interpreted programming language that allows developers to create dynamic and interactive web content.

It is widely used for web development, alongside HTML and CSS.

  • Client-side scripting
  • Server-side scripting (Node.js)
  • Supports object-oriented programming
console.log("Hello, JavaScript!");
Setup and Installation

JavaScript runs in web browsers, so no installation is required.

To include JavaScript in an HTML file, use the tag.

  <script>
    alert("Welcome to JavaScript!");
</script>
  • Use an external JavaScript file
  • Include scripts in the HTML file
  • Load scripts asynchronously
Syntax and Variables

JavaScript uses a simple syntax with variables, functions, and operators.

There are three ways to declare variables:

  • let (block-scoped, preferred)
  • const (constant values)
  • var (function-scoped, legacy)
// Declaring variables
let name = "Mahesh";
const age = 21;
var city = "Dhankuta";
Functions and Scope

Functions allow code reuse and modular programming.

They can be declared using function expressions or arrow functions.

// Function Declaration
function greet(name) {
    return "Hello, " + name;
}

// Arrow Function
const add = (a, b) => a + b;
  • Functions can return values
  • Functions can accept parameters
  • Arrow functions provide concise syntax
Events and DOM

JavaScript can manipulate the Document Object Model (DOM) to make web pages interactive.

Event listeners handle user actions like clicks, key presses, and form submissions.

// Selecting an element
const button = document.querySelector("button");

// Adding an event listener
button.addEventListener("click", function() {
    alert("Button clicked!");
});
  • Modify HTML content dynamically
  • Change CSS styles using JavaScript
  • Add event listeners for user interactions
